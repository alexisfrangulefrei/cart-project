# Cart Project

Small TypeScript cart project meant to practice Test-Driven Development (TDD).
The `Cart` model tracks product references, unit prices, and quantities while
enforcing the business rules encoded in the Vitest suite.

## Prerequisites

- Node.js 18+ (or any version compatible with Vitest)
- npm

## Install Dependencies

```bash
npm install
```

## Run the Test Suite

```bash
npm test
```
